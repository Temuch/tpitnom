## Задание 1: Платежный эквайринг университета

### Описание проекта

Система платежного эквайринга для университета - комплексное решение для приема различных видов платежей от студентов. Проект включает следующие ключевые компоненты:

- **Типы платежей**: оплата обучения, общежития, полиграфии, студенческого билета и других услуг
- **Методы оплаты**: банковские карты, СБП (Система быстрых платежей), SberPay
- **Техническая инфраструктура**: серверы, платежные шлюзы, электронные кассы
- **UX для студентов**: веб-интерфейс, мобильное приложение, личный кабинет, история платежей, автоплатежи
- **Интеграция**: связь с системами 1C, API, CRM, аналитика, административная панель
- **Финансовая часть**: поступления на счета (казначейский/внебюджетный), комиссии, отчетность
